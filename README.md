# pnp
A collection of tools for solving the Perspective-n-Point (PnP) problem in compute vision.

### compile and run
```bash
# get into folder
cd source_folder
# generate empty folder to store compilation file
mkdir build
# compile configuration
cmake ..
# compile
make install
# run
./epnp
```
